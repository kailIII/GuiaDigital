Descripción para archivo fig.006.gif

El diagrama representa el flujo de actividad de un mensaje con una serie de flujos alternativos. Comienza con la actividad "Envío de formulario"; pasa luego a "Recepción OIRS" con estado Pendiente; a continuación el flujo se abre en dos direcciones, la primera es responder automáticamente luego de 72 horas, la segunda es un hito de desición con la pregunta ¿enviar respuesta? Si no se envía respuesta se pasa el mensaje a estado "archivado". Si se decide enviar respuesta el mensaje se marca como "en trámite"; de ahí pasa a otra decisión con la pregunta ¿respuesta OK?. Continúa en este flujo hasta que la respuesta es positiva. En este punto se envía respuesta al usuario, se marca el mensaje como "respondido" y se genera una nueva desición: ¿es pregunta de interés general? Si lo es, se publica como Pregunta Frecuente, si no el mensaje se marca como "Archivado".
