Descripción para archivo fig.005.gif

La imagen presenta un diagrama de flujo que ilustra la interacción de un usuario para ingresar a una zona de acceso restringido. El detalle de la interacción es el siguiente:

  • Un usuario solicita el acceso a la portada
  • El sistema detecta si el usuario está o no registrado
  • Si el usuario está registrado le presenta contenidos personalizados y exclusivos para usuarios autorizados
  • Si el usuario no está registrado se le presentan los contenidos estándar accesibles para cualquier usuario
