Estándares de la Guía Web

El desarrollo de las páginas de este Sitio Web fue realizado para cumplir con las normas definidas en los estándares públicos recomendados y definidos por el World Wide Web Consortium, con el objetivo de que pueda ser visualizado sin problemas y usando cualquier tecnología de acceso.

Para ello, se trabajó utilizando los siguientes estándares:

XHTML 1.0 Transicional: Se eligió este estándar de construcción HTML para ordenar, estructurar y restringir el vocabulario de marcas (tags), mediante el uso de elementos semánticamente ricos sin contaminar el contenido con elementos presentacionales habituales de HTML tradicional. En este sentido, se ha privilegiado el uso de encabezados desde el rango H1 en adelante, pasando por tablas estructuradas, listados múltiples, abreviaciones y acrónimos. El resultado permite ver documentos prescindiendo del diseño y conservando su jerarquía y lógica de contenido.

CSS Nivel 2: Se empleó este estándar para diagramación y presentación porque separa efectivamente el contenido de la presentación grafica en pantalla. Gracias a esto, por ejemplo, este Sitio no ha sido diagramado usando tablas y, también, al imprimir cualquiera de sus páginas se cuenta con una nueva plantilla de presentación que adapta el contenido para formato impreso. Además, esta forma de trabajo permite revisar el Sitio adecuadamente desde cualquier tipo de browser.

WAI -AAA: Gracias al uso de XHTML y CSS, este sitio cumple satisfactoriamente con las especificaciones del más alto nivel de Accesibilidad definido por el World Wide Web Consortium, asegurando que personas con discapacidades podrán acceder sin dificultades a todo su contenido. Un ejemplo concreto de esto, es el uso del atributo LONGDESC para explicar en texto el contenido de una imagen. En este sentido, hay que agregar que además de las validaciones automáticas existentes, hay un conjunto de otras, de tipo manual, que este sitio se ha encargado de cumplir adecuadamente.

A continuación, se presentan los íconos que demuestran que este Sitio Web ha aprobado cada uno de los test de los estándares que han sido mencionados en esta Guía. Al hacer click sobre ellos, es posible acceder al sistema de validación externo para comprobar el cumplimiento de dichos estándares.
