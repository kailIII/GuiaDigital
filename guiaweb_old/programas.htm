Tipos de Archivos y Programas Auxiliares

La Guía Web contiene documentos en diferentes formatos de archivo, por lo que necesitará algunos programas auxiliares para poder verlos apropiadamente. A continuación le informaremos sobre los formatos y los programas asociados y los íconos usados en el sitio para identificarlos.

Tipo de Archivo Extensión Programa Asociado Comentarios
HTML, HyperText Markup Language htm, html (otras) MS Internet Explorer, Firefox, Netscape, Mozilla, Opera, Safari, otros. Este tipo de archivos es el más común en el web y podrá verlo sin ningún programa adicional aparte de su browser. Si está viendo esta página podrá ver cualquier documento HTML.
PDF, Portable Document Format pdf Acrobat Reader Descargue gratis Acrobat Reader desde el sitio de Adobe.
Microsoft Word doc MS Word Para ver documentos en formato Word deberá tener instalado MS Word. Si no lo tiene, puede descargar un visualizador de documentos Word gratuíto desde el sitio de Microsoft.
RTF, Rich Text Format rtf MS Word, WordPad, OpenOffice Writer, otros. Rich Text Format o RTF es un tipo de archivo de texto de formato enriquecido creado por Microsoft, es muy común y puede ser leído o creado con diversos programas y en distintos sistemas operativos.
Archivo comprimido ZIP zip WinZip, PKZIP, WinRAR Un documento ZIP es un archivo comprimido para utilizar un espacio menor y facilitar su almacenamiento y descarga.
Excel xls MS Excel, OpenOffice Spreadsheet, otros. Para ver documentos en formato Excel deberá tener instalado MS Excel. Si no lo tiene, puede descargar un visualizador de documentos Excel gratuíto desde el sitio de Microsoft.
