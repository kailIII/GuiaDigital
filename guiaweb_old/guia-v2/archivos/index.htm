Archivos para descargar de la Guía Web - Versión 2.0

En esta sección se ofrecen los archivos en formato PDF de cada uno de los capítulos de esta Guía, como también otros documentos que son referenciados en los propios capítulos.

Capítulos en Formato PDF

  • Capítulo 0: Introducción a esta Versión - PDF: 3065 Kb. Este capítulo explica los alcances de la Guía Web, sus características y contenidos y su relación tanto con su propio Sitio Web, como con la Guía Web Versión 1.0 (Incluye diseño de la portada). En el presente Sitio Web este capítulo lleva el nombre de Introducción.
  • Capítulo 1: Normas que rigen los Sitios Web - PDF: 111 Kb. Este capítulo se refiere a las normas oficiales del Gobierno que deben ser cumplidas por los Sitios Web. En el presente Sitio Web este capítulo lleva el nombre de Normativa.
  • Capítulo 2: Aplicación de Estándares - PDF: 293 Kb. Este capítulo se refiere a cuáles son los estándares que debe cumplir el Sitio Web, haciendo especial referencia a lo señalado en los decretos 81 y 100 que norman estos aspectos. En el presente Sitio Web este capítulo lleva el nombre de Estándares.
  • Capítulo 3: Diseño de Interfaces e Interacción - PDF: 832 Kb. Este capítulo explica de manera práctica la forma de ejecutar la separación de presentación y contenidos. En el presente Sitio Web este capítulo lleva el nombre de Diseño Web.
  • Capítulo 4: Cómo se llega al Sitio Web - PDF: 252 Kb. Este capítulo introduce el concepto de la Encontrabilidad como elemento clave para asegurar que sus contenidos queden adecuadamente indexados en sistemas de búsqueda internos y externos, facilitando de esa manera el acceso directo a la información. En el presente Sitio Web este capítulo lleva el nombre de Encontrabilidad.
  • Capítulo 5: De la Usabilidad a la Utilidad - PDF: 271 Kb. Este capítulo introduce el concepto de la Usabilidad como elemento central para los Sitios Web de Gobierno, desde el punto de vista de la forma de presentar los contenidos. Incluye indice general de la Guía Web 2.0. En el presente Sitio Web este capítulo lleva el nombre de Usabilidad.
  • Archivo que incluye todos los Capítulos de la Versión 2.0 de la Guía Web - PDF: 4622 Kb.

Otros documentos ofrecidos en el Sitio Web

En esta sección se ofrecen otros archivos en formato PDF que se ofrecen en los diferentes capítulos de esta Guía.

  • Capítulo 2: Diagramación de texto con CSS. Prueba.htm es un archivo de ejemplo para ver la forma de aplicar las etiquetas <div> para dar formato a un texto. Este archivo se referencia en la sección Puesta en Marcha
  • Capítulo 2: Archivo de Contenido para uso con CSS. 3cols.htm consiste en una página con tres columnas: una tiene un logotipo y las otras dos, texto simulado; en su código se puede ver que sólo tiene etiquetas de html estándar. Este archivo se referencia en la sección Utilización de las Hojas de Estilo (CSS)
  • Capítulo 2: Hojas de Estilo CSS. 3cols.css y 3cols-print.css se llaman desde la sección <head> de su código; una de ellas es para mostrar el contenido en pantalla, mientras que la segunda es para impresión. Este archivo se referencia en la sección Utilización de las Hojas de Estilo (CSS)
  • Capítulo 5: Pauta de Evaluación Heurística. Pauta-evaluacion-heuristica.pdf es un documento que muestra cómo utilizar esta metodología para asegurar la usabilidad de un sitio web. El documento puede emplearse como modelo para este tipo de tests.
  • Capítulo 5: Pauta de Test de Usuario. Pauta-test-usuario.pdf contiene los elementos de evaluación que se pueden utilizar para hacer la evaluación de un sitio web.
  • Capítulo 5: Muestra gráfica de una Persona y Escenario. Pauta-persona-escenario.pdf que puede emplearse como modelo para implementar esta metodología en forma práctica.

Cómo obtener el lector de archivos PDF

Para utilizar los archivos en formato PDF debe emplear un software adicional llamado "Acrobat Reader" que permite utilizar los archivos en formato PDF. Si no lo tiene, puede bajarlo gratuitamente desde el sitio de Adobe.
