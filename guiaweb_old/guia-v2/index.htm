Contenidos de la Guía Web - Versión 2.0

Aquí encontrará la versión en formato HTML de cada uno de los capítulos de esta Guía, más las copias digitales (en formato PDF).

Adicionalmente, en cada uno de los capítulos se entregarán, cuando corresponda, los anexos indicados en sus textos.

Capítulo 0: Introducción a esta Versión

Este capítulo explica los alcances de la Guía Web, sus características, contenidos y su relación tanto con su propio Sitio Web, como con la Guía Web Versión 1.0.

En el presente Sitio Web este capítulo lleva el nombre de Introducción.

Capítulo 1: Normas que rigen los Sitios Web

Este capítulo se refiere a las normas oficiales del Gobierno que deben ser cumplidas por los Sitios Web, en las que se establecen los contenidos mínimos que deben cumplir los Sitios Web para su operación; en particular se hará referencia al cumplimiento de los Instructivos Presidenciales más las leyes vigentes en estos aspectos.

En el presente Sitio Web este capítulo lleva el nombre de Normativa.

Capítulo 2: Aplicación de Estándares

Este capítulo se refiere a cuáles son los estándares que debe cumplir el Sitio Web, haciendo especial referencia a lo señalado en los decretos 81 y 100 que norman estos aspectos. Entre ellos se contarán los referidos a las normas de estructura de documentos y accesibilidad del World Wide Web Consortium (W3C) y las relativas a interoperabilidad establecidas por el Decreto 81.

En el presente Sitio Web este capítulo lleva el nombre de Estándares.

Capítulo 3: Diseño de Interfaces e Interacción

Este capítulo se refiere a la forma en que se presentan los contenidos en los Sitios Web; el objetivo de este capítulo es explicar de manera práctica la forma de ejecutar la separación de presentación y contenidos, en particular utilizando las tecnologías descritas en el Decreto 100.

En el presente Sitio Web este capítulo lleva el nombre de Diseño Web.

Capítulo 4: Cómo se llega al Sitio Web

Este capítulo introduce el concepto de la Encontrabilidad (entendido como la facilidad para que los contenidos de un Sitio Web puedan ser indexados y luego encontrados por sistemas de búsqueda externos e internos) como elemento clave para asegurar que sus contenidos queden adecuadamente indexados en sistemas de búsqueda internos y externos, facilitando de esa manera el acceso directo a la información. La cobertura de este aspecto estará relacionada con la forma de generar, escribir y presentar los contenidos.

En el presente Sitio Web este capítulo lleva el nombre de Encontrabilidad.

Capítulo 5: De la Usabilidad a la Utilidad

Este capítulo introduce el concepto de la Usabilidad (entendido como facilidad de uso) como elemento central para los Sitios Web de Gobierno, poniendo énfasis en la necesidad de que se asegure que lo que se publica pueda ser visto y usado sin problemas de acceso por los ciudadanos. La cobertura de este aspecto estará relacionada con la forma de presentar los contenidos.

En el presente Sitio Web este capítulo lleva el nombre de Usabilidad.
